# Roadmap

This document outlines the planned evolution of the Financial Report Analysis Agent.

---

## Phase 1: Local analysis engine (current)

- CLI-based interface
- Single-document ingestion
- Citation-grounded summaries
- Structured metric extraction
- Evidence-based Q&A

---

## Phase 2: Backend platform

- FastAPI service layer
- Background job processing
- Multi-document reasoning
- Improved table extraction
- Persistent storage

---

## Phase 3: Web application

- Web-based UI
- Document upload and management
- Saved analyses
- Company and period comparisons
- User authentication

---

## Phase 4: Advanced analysis

- Industry-specific templates
- Red flag detection
- Trend analysis across periods
- Analyst-style commentary generation
